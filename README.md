# Matasaa.co.za

Pre-reserved transportation service website - Static site hosted on AWS CloudFront

## Architecture

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Hosting**: AWS S3 + CloudFront (af-south-1 Cape Town region)
- **Edge Location**: CPT51-P1 (Cape Town)
- **Backend API**: Tredicik Platform
- **DNS**: Cloudflare
- **SSL**: AWS Certificate Manager (ACM)
